この度は、拡張にゃしぃフレンズをダウンロードしていただき、
ありがとうございます。



1. 拡張にゃしぃフレンズとは
拡張にゃしぃフレンズは、
にゃしぃフレンズ（圧縮ファイル同封）の字形を
文字通り「拡張」するフォントです。

フォント制作における制約上、
これまで「メイン（＝にゃしぃフレンズ）」の
フォントファイルに収録しようとしても、できなかった漢字たち。
それらを救済すべく、拡張にゃしぃフレンズはメインに対する
「サブ」のフォントファイルという位置づけで、
約300字（ひらがな・カタカナ・漢字）を収録。
いわゆる「下長の吉」の文字データを「上長の吉」の領域に入れ込むなど、
よく似た文字の領域に異体字などを組み入れています。

人名用漢字の異体字や、印刷標準字体にも対応。
にゃしぃフレンズと併せてインストールしていただき、
必要に応じて使い分けていただければと思います。

※変換割り当てについては、
添付ファイル「収録文字一覧.txt」をご覧ください。



2. 収録文字
・ひらがな…ガ行鼻濁音を示す5字
・カタカナ…ガ行鼻濁音を示す5字
・アルファベット…なし
・数字…なし
・記号…59種類
・漢字…人名用漢字の異体字や、印刷標準字体など、261字



3. 動作環境
現在のところ、
Microsoft Windows XP、Vista、7、8、10で動作を確認しております。



4. ご使用・配布について
ウェブサイトや印刷物、動画などでの使用のほか、
商用利用（同人誌などでの使用）についても、
特に制限は設けておりません。
ご自由にお使いいただければ制作者冥利に尽きます。

使用許可連絡も必要ございませんが、
よろしければ事後でも結構ですので、
web拍手やTwitterなどでご一報いただければありがたいです。

なお、字形あるいは著作権表示、
もしくはその両方を改変しての二次配布はご容赦くださいませ。
